API Project: File Metadata Microservice

Upload a file and get back its metadata!

File Upload



API Information

Endpoint: POST /api/fileanalyse

Method: Upload any file using the form above

Returns: JSON object with file metadata

Example Response

{ "name": "example.pdf", "type": "application/pdf", "size": 1024567 }

Response Fields:

• name: Original filename

• type: MIME type of the file

• size: File size in bytes

Usage Instructions

1. Click "Choose File" to select any file from your computer

2. Click "Upload File" to submit the file

3. View the JSON response with file metadata

Note: Files are not stored on the server - only metadata is analyzed and returned.
